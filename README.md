# Al Alameya for Contracting & General Supplies

Production-ready PHP 8 MVC corporate website designed for free shared hosting (AeonFree) with no Composer and no framework dependencies.

## Stack
- PHP 8 (OOP, custom MVC)
- MySQL + PDO prepared statements
- Bootstrap 5 + Vanilla JavaScript

## Features
- Public pages: Home, About, Services, Projects, Blog, Careers, Contact
- Admin panel with role-based access (Admin/Editor)
- CRUD: Projects, Blog, Services, Users
- Security: CSRF, XSS escaping, password hashing, session regeneration, upload validation
- SEO: clean URLs, robots.txt, sitemap.xml, dynamic meta tags
- Performance: file cache + lazy loaded images

See `DEPLOYMENT_AEONFREE.md` for setup instructions.
