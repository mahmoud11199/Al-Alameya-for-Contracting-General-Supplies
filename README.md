# Al Alameya for Contracting & General Supplies

Production-ready PHP 8 MVC corporate website for AeonFree free hosting (no Composer, no Laravel, no CLI dependencies after upload).

## Stack
- PHP 8 (OOP, custom MVC)
- MySQL + PDO prepared statements
- Bootstrap 5 + Vanilla JavaScript

## Features
- Public pages: Home, About, Services, Projects, Blog, Careers, Contact
- Admin panel with role-based access (Admin/Editor)
- CRUD: Projects, Blog, Services, Users
- Security: CSRF, output escaping, password hashing, session regeneration, upload validation, basic rate-limits
- SEO: clean URLs, robots.txt, sitemap.xml, dynamic meta tags
- Performance: lightweight file cache, lazy-loaded images, DB indexes for common queries
